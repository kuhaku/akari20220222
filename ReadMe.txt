
神岸あかり浣腸20220222 (あかかんにゃんにゃんにゃん)

世紀を超えて勝手に蘇った時代錯誤の糞ゲー！

君はバッグいっぱいに浣腸を隠し持った変質者だ。神岸あかりに浣腸し、うんこさせてやろう。
ただし、あかりは必死で抵抗してくるはずだ。うまく押さえつけて、浣腸してやろう。

＜ゲームの進め方＞
変質者は矢印キーで動く。
Zキーで無花果浣腸を発射出来るぞ。ただし無花果浣腸の威力はそのままでは弱い。
そこで、こすり (東○でいうところのGRAZE、サイ○○リアでいうところのBUZZ) システムだ！
弾にこするほど攻撃力がアップするぞ。

さりとて、あかりも必死に抵抗する。もたもたしていると防犯ブザーを鳴らし
権力に飼い慣らされた犬どもを呼んでブタ箱行きにされるぞ。


＜アイテムの説明＞

エネマシリンジ：
変質者の攻撃力が上がる。

ToHeart：
変質者の体力が1アップ。
とにかく取れ！タフすぎて損はない。

いつまで経っても続編が出ないので勝手に作りました。
オリジナルとは別人の元あやしい住人です。
同じゲームを繰り返しプレイさせるには、
スコア・タイム・収集といった要素が必要になるかと思いますが、
糞ゲーなのでそんなものはありません。
いかに早く作ることだけを考えました。
といったわけで雑ですが、
生まれて初めて作ったゲームなので
少しでも楽しんでいただけれたらと。
